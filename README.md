# devsecopstwo

## Сначала был код:

```
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello world, hi!");
    }
}
```
## Потом код был исправлен на:
```
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello world, hi!");
    }
}
```
и обновлен с помомощью v1.0.0
